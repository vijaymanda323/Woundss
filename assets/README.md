# Assets Directory

This directory contains the app icons and splash screens for the React Native application.

## Required Assets

### App Icons
- `icon.png` - Main app icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)

### Splash Screen
- `splash.png` - Splash screen image (1284x2778)

### Web Assets
- `favicon.png` - Web favicon (32x32)

## Creating Assets

You can create these assets using:
1. **Design Tools**: Figma, Adobe XD, Sketch
2. **Online Generators**: Expo Icon Generator, App Icon Generator
3. **AI Tools**: DALL-E, Midjourney for generating medical-themed icons

## Asset Specifications

### App Icon
- **Size**: 1024x1024 pixels
- **Format**: PNG
- **Background**: Transparent or solid color
- **Design**: Medical/wound healing theme

### Splash Screen
- **Size**: 1284x2778 pixels (iPhone 12 Pro Max resolution)
- **Format**: PNG
- **Background**: Gradient matching app theme (#667eea to #764ba2)
- **Content**: App logo and loading indicator

### Favicon
- **Size**: 32x32 pixels
- **Format**: PNG
- **Design**: Simplified version of app icon

## Temporary Assets

For development and testing, you can use placeholder images or generate simple assets using online tools. The app will work without these assets, but they're required for production builds.

- **Expo Icon Generator**: https://docs.expo.dev/guides/app-icons/
